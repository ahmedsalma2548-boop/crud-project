# CRUD Project

A simple CRUD (Create, Read, Update, Delete) application built to manage data efficiently.

## 🚀 Live Demo
👉 https://crud-arabic-app.netlify.app/

---

## ✨ Features
- Add new items  
- Display all items  
- Edit existing items  
- Delete items  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

---

## 📂 How to Run the Project
1. Clone the repository  
2. Open the project folder  
3. Run `index.html` in your browser
